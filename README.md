# Algorithms Java

Estructuras de datos, algoritmos de búsqueda, programación dinámica, optimización y benchmarks con JMH.

## Proyectos

| Carpeta | Unidad | Descripción | Tema |
|---------|--------|-------------|------|
| `u2-sorting-benchmarks/` | U2 | Sorting algorithms + ArrayList vs LinkedList + cache locality | Sorting, análisis empírico |
| `u3-fibonacci/` | Post1 U3 | Fibonacci: naive vs memoization vs tabulation + JMH | DP básico, benchmarks |
| `u3-huffman-nqueens/` | Post2 U3 | Huffman Encoding + N-Queens (Naive vs Bitmask) | Compresión, backtracking |
| `u4-bloom-filter/` | Post1 U4 | Bloom Filter vs HashSet + JMH | Estructuras probabilísticas |
| `u4-avl-dsu/` | Post2 U4 | AVL Tree + DSU (Union-Find) + JMH | Árboles, grafos |
| `u6-dp-knapsack/` | Post1 U6-2026 | Knapsack 0/1 + Weighted Scheduling + JMH | DP, optimización |
| `u7-binary-search-kmp/` | Post1 U7-2026 | Binary Search + KMP + Suffix Array + JMH | Búsqueda, strings |
| `u6-lcs-floyd/` | Post2 U6-2026 | LCS + Edit Distance + Floyd-Warshall + JMH | DP strings, grafos |
| `u7-tfidf-sparse-segment-fenwick/` | Post2 U7-2026 | TF-IDF + SparseTable + SegmentTree + Fenwick + JMH | IR, range queries |
| `u8-lru-cache/` | Post2 U8-2026 | LRU Cache + WriteThrough + WriteBehind + OptimisticLock + JMH | Caching, concurrencia |

## Tecnologías

- Java 17+
- JMH (Java Microbenchmark Harness)
- Maven
